# ­ЪЊџ рЃџрЃћрЃЦрЃфрЃўрЃљ 1 РђЊ рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљрЃерЃў рЃЊрЃљ Python-рЃўрЃА рЃљрЃќрЃарЃЮрЃЋрЃюрЃћрЃЉрЃљрЃерЃў

**рЃЏрЃўрЃќрЃљрЃюрЃў:** рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃЏрЃљ рЃњрЃљрЃўрЃњрЃЮрЃю рЃарЃљ рЃљрЃарЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљ, рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА Python рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃћрЃюрЃљ рЃЊрЃљ рЃерЃћрЃФрЃџрЃЮрЃю рЃЌрЃљрЃЋрЃўрЃљрЃюрЃЌрЃў рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЎрЃЮрЃЊрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ CS50 VS Code рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљрЃќрЃћ.

## ­Ъј» рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃЏрЃўрЃќрЃюрЃћрЃЉрЃў

-   рЃљрЃ«рЃАрЃюрЃљрЃю рЃарЃљ рЃљрЃарЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃў рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ.
-   рЃЏрЃўрЃ«рЃЋрЃЊрЃюрЃћрЃю рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА Python рЃърЃЮрЃърЃБрЃџрЃљрЃарЃБрЃџрЃў рЃЊрЃљ рЃЊрЃљрЃЏрЃгрЃДрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЏрЃЮрЃАрЃљрЃ«рЃћрЃарЃ«рЃћрЃЉрЃћрЃџрЃў.
-   рЃерЃћрЃФрЃџрЃЮрЃю CS50 VS Code рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљрЃќрЃћ рЃЎрЃЮрЃЊрЃўрЃА рЃгрЃћрЃарЃљ рЃЊрЃљ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ.
-   рЃњрЃљрЃљрЃарЃЕрЃўрЃЮрЃю **рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃўрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ** рЃЊрЃљ **рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ**.
-   рЃЊрЃљрЃгрЃћрЃарЃЮрЃю рЃЌрЃљрЃЋрЃўрЃљрЃюрЃЌрЃў рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ Python-рЃерЃў.

## ­ЪДа рЃЌрЃћрЃЮрЃарЃўрЃљ

### 1. рЃарЃљ рЃљрЃарЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљ?

-   рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљ рЃљрЃарЃўрЃА рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃЊрЃљрЃгрЃћрЃарЃљ, рЃарЃљрЃЌрЃљ рЃЏрЃљрЃю рЃерЃћрЃљрЃАрЃарЃБрЃџрЃЮрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ.
-   рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃљ: рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃљрЃарЃўрЃА рЃарЃћрЃфрЃћрЃърЃбрЃў, рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃў РђЊ рЃЏрЃќрЃљрЃарЃћрЃБрЃџрЃў.

### 2. рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃћрЃюрЃћрЃЉрЃў

-   рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃЕрЃљрЃАрЃљрЃгрЃћрЃарЃў РђърЃћрЃюрЃћрЃЉрЃўРђю.
-   рЃърЃЮрЃърЃБрЃџрЃљрЃарЃБрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў: **Python**, JavaScript, C, Java.
-   Python рЃљрЃарЃўрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃљ рЃЎрЃўрЃЌрЃ«рЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА рЃљрЃЊрЃЋрЃўрЃџрЃў рЃћрЃюрЃљ РђЊ рЃўрЃЊрЃћрЃљрЃџрЃБрЃарЃў рЃЊрЃљрЃАрЃљрЃгрЃДрЃћрЃЉрЃљрЃЊ.

### 3. рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃў

-   рЃюрЃљрЃЉрЃўрЃ»-рЃюрЃљрЃЉрЃўрЃ» рЃњрЃћрЃњрЃЏрЃљ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃўрЃА рЃњрЃљрЃЊрЃљрЃАрЃљрЃГрЃарЃћрЃџрЃљрЃЊ.
-   рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў: рЃЕрЃљрЃўрЃА рЃЏрЃЮрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃљ Рєњ рЃгрЃДрЃџрЃўрЃА рЃљрЃЊрЃБрЃдрЃћрЃЉрЃљ Рєњ рЃЕрЃљрЃўрЃА рЃЕрЃљрЃДрЃарЃљ Рєњ рЃЊрЃљрЃџрЃЮрЃЊрЃћрЃЉрЃљ Рєњ рЃЊрЃљрЃџрЃћрЃЋрЃљ.

### 4. рЃарЃљрЃбрЃЮрЃЏ Python?

-   рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃў (РђърЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉ рЃарЃЮрЃњрЃЮрЃарЃф рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃа рЃбрЃћрЃЦрЃАрЃбрЃАРђю)
-   рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ:
    -   рЃЋрЃћрЃЉ-рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃбрЃерЃў
    -   рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЏрЃћрЃфрЃюрЃўрЃћрЃарЃћрЃЉрЃљрЃерЃў
    -   рЃ«рЃћрЃџрЃЮрЃЋрЃюрЃБрЃа рЃўрЃюрЃбрЃћрЃџрЃћрЃЦрЃбрЃерЃў
    -   рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃўрЃќрЃљрЃфрЃўрЃљрЃерЃў
-   рЃЊрЃўрЃЊрЃў рЃАрЃљрЃќрЃЮрЃњрЃљрЃЊрЃЮрЃћрЃЉрЃљ рЃЊрЃљ рЃБрЃљрЃЏрЃарЃљрЃЋрЃў рЃарЃћрЃАрЃБрЃарЃАрЃў.

### 5. рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃАрЃўрЃЋрЃарЃфрЃћ РђЊ CS50 VS Code

-   рЃЋрЃћрЃЉ-рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљ, рЃАрЃљрЃЊрЃљрЃф рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃЎрЃЮрЃЊрЃўрЃА рЃгрЃћрЃарЃљ.
-   рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ рЃљрЃа рЃљрЃарЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ.
-   рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ: `mkdir my_folder`
-   рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃерЃў рЃерЃћрЃАрЃЋрЃџрЃљ: `cd my_folder`
-   рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ: `code my_code.py`
-   рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ: `python my_code.py`
-   рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃўрЃЊрЃљрЃю рЃњрЃљрЃЏрЃЮрЃАрЃЋрЃџрЃљ: `cd ..`

## ­Ъњ╗ рЃџрЃљрЃўрЃЋ рЃЎрЃЮрЃЊрЃўрЃюрЃњрЃў

### рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ РђЊ Hello World

```python
print("Hello, world!")
```



# ­Ъњ╗ рЃџрЃћрЃЦрЃфрЃўрЃљ 2 РђЊ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃљ: рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃў Python-рЃерЃў

**рЃЏрЃўрЃќрЃљрЃюрЃў:** рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃЏрЃљ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃљрЃЊ рЃњрЃљрЃюрЃљрЃЏрЃбрЃЎрЃўрЃфрЃЮрЃю рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃџрЃћрЃЦрЃфрЃўрЃўрЃА рЃфрЃЮрЃЊрЃюрЃљ Рђћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃю `print()` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ рЃбрЃћрЃЦрЃАрЃбрЃўрЃАрЃљ рЃЊрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЏрЃљрЃЌрЃћрЃЏрЃљрЃбрЃўрЃЎрЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃбрЃљрЃюрЃљрЃЊ, рЃўрЃАрЃгрЃљрЃЋрЃџрЃЮрЃю рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃърЃЮрЃЋрЃюрЃљ (debugging) рЃЊрЃљ рЃерЃћрЃЦрЃЏрЃюрЃљрЃю рЃЌрЃљрЃЋрЃўрЃљрЃюрЃЌрЃў рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃърЃљрЃбрЃљрЃарЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃў.

---

## ­Ъј» рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃЏрЃўрЃќрЃюрЃћрЃЉрЃў

-   рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃю `print()` рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃцрЃЮрЃарЃЏрЃўрЃЌ рЃбрЃћрЃЦрЃАрЃбрЃўрЃАрЃљ рЃЊрЃљ рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃбрЃљрЃюрЃљрЃЊ.
-   рЃњрЃљрЃўрЃњрЃЮрЃю рЃарЃљрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА рЃљрЃ«рЃљрЃџрЃў рЃ«рЃљрЃќрЃў `\n` рЃЊрЃљ рЃбрЃљрЃЉрЃБрЃџрЃљрЃфрЃўрЃљ `\t`.
-   рЃўрЃАрЃгрЃљрЃЋрЃџрЃЮрЃю рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃљрЃдрЃЏрЃЮрЃЕрЃћрЃюрЃљ рЃЊрЃљ рЃњрЃљрЃАрЃгрЃЮрЃарЃћрЃЉрЃљ (syntax / runtime errors).
-   рЃерЃћрЃЦрЃЏрЃюрЃљрЃю рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃў (`*`-рЃўрЃЌ рЃюрЃљрЃ«рЃљрЃбрЃћрЃЉрЃў, рЃАрЃљрЃ«рЃћрЃџрЃўрЃА рЃЕрЃљрЃарЃЕрЃЮ рЃЊрЃљ рЃАрЃ«рЃЋ.)
-   рЃњрЃљрЃюрЃўрЃЋрЃўрЃЌрЃљрЃарЃЮрЃю рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃљрЃќрЃарЃЮрЃЋрЃюрЃћрЃЉрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃўрЃЌ.

---

## ­ЪДа рЃЌрЃћрЃЮрЃарЃўрЃљ рЃЏрЃЮрЃЎрЃџрЃћрЃЊ (рЃарЃћрЃърЃћрЃбрЃўрЃфрЃўрЃљ)

-   `print()` рЃљрЃарЃўрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃћрЃЎрЃарЃљрЃюрЃќрЃћ рЃњрЃљрЃЏрЃЮрЃбрЃљрЃюрЃА рЃбрЃћрЃЦрЃАрЃбрЃА рЃљрЃю рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃџрЃў рЃерЃћрЃЊрЃћрЃњрЃА.
-   рЃбрЃћрЃЦрЃАрЃбрЃў рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃўрЃгрЃћрЃарЃћрЃЉрЃљ рЃЉрЃарЃГрЃДрЃљрЃџрЃћрЃЉрЃерЃў `" "` рЃљрЃю `' '`.
-   Python рЃЎрЃЮрЃЊрЃА рЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉрЃА рЃќрЃћрЃЏрЃЮрЃЊрЃљрЃю рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ РђЊ рЃћрЃарЃЌрЃў рЃ«рЃљрЃќрЃў рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЏрЃћрЃЮрЃарЃћрЃА.
-   рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА:
    -   **SyntaxError** РђЊ рЃарЃЮрЃфрЃљ рЃЎрЃЮрЃЊрЃў рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃљрЃЊ рЃљрЃарЃўрЃА рЃЊрЃљрЃгрЃћрЃарЃўрЃџрЃў (рЃЏрЃљрЃњ. рЃњрЃљрЃЏрЃЮрЃбрЃЮрЃЋрЃћрЃЉрЃБрЃџрЃў рЃЉрЃарЃГрЃДрЃљрЃџрЃў)
